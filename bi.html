Business Performance Dashboard Overview

CPI Card Group is a credit card manufacturer that has recently started selling EMV credit cards. I created this business intelligence tool to track manufacturing metrics to help identify issues and inform process improvement decisions. It currently has one dashboard for cost performance tracking. More to come.


Manufacturing Cost Performance

The dashboard shown below in figure 1 represents the manufacturing cost performance for the EMV credit cards by card type, location, cost type, year, and machine model. The BI is connected to a SQL server data warehouse and runs on a native query. Budgets are calculated by dollars per unit budgeted cost multipled by actual units, multiple by actual units built.
